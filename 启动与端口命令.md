# 启动与端口命令

适用于当前 Windows + PowerShell 环境。

## 进入项目目录

```powershell
cd E:\Moona
```

## 安装依赖

首次运行或依赖变化后执行：

```powershell
npm install
```

## 启动后端 API

```powershell
npm --workspace apps/api run start:dev
```

默认地址：

```text
http://localhost:3000
```

## 启动前端 H5

```powershell
npm --workspace apps/mobile run dev:h5
```

默认地址：

```text
http://localhost:5173/index.html
```

## 同时启动前后端

分别打开两个 PowerShell 窗口执行：

```powershell
cd E:\Moona
npm --workspace apps/api run start:dev
```

```powershell
cd E:\Moona
npm --workspace apps/mobile run dev:h5
```

## 后台启动前后端

```powershell
Start-Process -FilePath npm -ArgumentList @('--workspace','apps/api','run','start:dev') -WorkingDirectory 'E:\Moona' -WindowStyle Hidden
Start-Process -FilePath npm -ArgumentList @('--workspace','apps/mobile','run','dev:h5') -WorkingDirectory 'E:\Moona' -WindowStyle Hidden
```

## 查询端口占用

查询后端 3000：

```powershell
netstat -ano | Select-String ':3000'
```

查询前端 5173：

```powershell
netstat -ano | Select-String ':5173'
```

同时查询常用端口：

```powershell
netstat -ano | Select-String ':3000|:5173|:5174|:5175'
```

## 查看占用端口的进程

先通过 `netstat` 找到最后一列 PID，然后执行：

```powershell
Get-Process -Id 进程PID
```

示例：

```powershell
Get-Process -Id 26120
```

## 杀掉指定 PID

```powershell
Stop-Process -Id 进程PID -Force
```

示例：

```powershell
Stop-Process -Id 26120 -Force
```

## 按端口杀进程

杀掉占用 3000 的进程：

```powershell
$pid = (Get-NetTCPConnection -LocalPort 3000 -State Listen).OwningProcess
Stop-Process -Id $pid -Force
```

杀掉占用 5173 的进程：

```powershell
$pid = (Get-NetTCPConnection -LocalPort 5173 -State Listen).OwningProcess
Stop-Process -Id $pid -Force
```

## 重启当前项目服务

```powershell
Get-CimInstance Win32_Process -Filter "name = 'node.exe'" |
  Where-Object { $_.CommandLine -like '*E:\Moona*' } |
  ForEach-Object { Stop-Process -Id $_.ProcessId -Force }

Start-Process -FilePath npm -ArgumentList @('--workspace','apps/api','run','start:dev') -WorkingDirectory 'E:\Moona' -WindowStyle Hidden
Start-Process -FilePath npm -ArgumentList @('--workspace','apps/mobile','run','dev:h5') -WorkingDirectory 'E:\Moona' -WindowStyle Hidden
```

## 数据库迁移

修改 Prisma 模型后执行：

```powershell
npm --workspace apps/api run prisma:migrate
```

只重新生成 Prisma Client：

```powershell
npm --workspace apps/api run prisma:generate
```

## 检查项目

类型检查：

```powershell
npm --workspace apps/api run typecheck
npm --workspace apps/mobile run typecheck
```

构建检查：

```powershell
npm run build
```
